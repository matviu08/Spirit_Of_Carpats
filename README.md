# Spirit of Carpats — Офіційний вебсайт

Цей репозиторій містить код офіційного сайту-візитки для інді-гри **"Spirit of Carpats"**. Сайт створений для презентації проєкту, розкриття лору гри та інформування гравців про етапи розробки.

## 🌲 Про гру "Spirit of Carpats"
**"Spirit of Carpats"** — це атмосферний 2D піксель-арт проєкт з яскраво вираженими паранормальними елементами. 

Головний герой — колишній лісник, який опиняється віч-на-віч із незвіданим у гущавині Карпатських гір. Гравцям доведеться досліджувати таємничі локації, розгадувати загадки карпатських лісів та виживати серед містичних явищ, які приховують ці стародавні місця.

## 💻 Технології
Вебсайт розроблено з використанням наступних технологій:
* **HTML5** — семантична структура сторінок.
* **CSS3** — стилізація, анімації та адаптивний дизайн (з підтримкою світлої/темної теми).
* **JavaScript** — інтерактивні елементи та логіка сайту.
